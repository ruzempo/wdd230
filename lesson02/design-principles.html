PARC: Proximity

UNITEC university

https://www.unitec.mx/licenciaturas/

Proximity is employed when there are multiple components close to each other. They become a single visual unit. So this principle organizes the information, reduces clutter and offers the reader a clear structure.

In the UNITEC university website we can see an example of proximity. The information is organized and contained in cards every one is close to each other, they become a unique visual unity to reduce shambles.

PARC: Repetition

Adele website

https://www.adele.com/

Repetition utilizes the same component repeatedly and is distributed throughout the web design. Usually, repetition is used in compositions of catalogs, magazines, personal branding, etc.

An example of the principle of repetition is the singer Adel's website. She uses through the page the same combination of text colors (withe and gold) and even uses the same image pattern with menus and text.

PARC: Contrast

Mercado Libre (online store)

https://www.mercadolibre.com.mx/

Contrast uses different shapes, colors, and fonts to highlight the most crucial elements to contrast with secondary elements. Contrasting colors, textures, sizes, and fonts create an intense visual effect and direct engagement to the main message in the website.

In the image above, the Mercado Libre website uses contrast to highlight the offers using a yellow background color with navy fonts with different font sizes.
